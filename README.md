# Portfolio

This is my portfolio website created with nextjs

## Getting Started

First, run the development server and open [http://localhost:3000](http://localhost:3000) with your browser to see the result:

```bash
pnpm dev
```

Build the project with:

```bash
pnpm build
```

And serve the built static files with:

```bash
pnpm serve
```
